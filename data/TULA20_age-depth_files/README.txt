README

tula20_agedepthmodel - list that contains
	- tula20_newChron = Bchronology output (Bchron file) 
	- tula20_chronPredicts = raw age predictions for median, 5th and 95th quantiles 
	- tula20_composite = table of composite core with core IDs and depth and composite core depths and new ages

tula20_CompositeCore_w-ages = csv version of tula20_composite above

TULA20_chronPlot = chronology plot output from Bchron

TULA20_controls = csv file containing all the chronology controls and associated chronIDs

*note in the following accu_rate is the sediment accumulation rate
TULA20_CFS_w-ages = .csv and .RData files with CFS updated ages
TULA20_LOI_w-ages = .csv and .RData files with LOI updated ages
TULA20_CHAR_w-ages = .csv and .RData files with CHAR updated ages, includes both counts and charcoal accumulation rate (CHAR) calcuated from the sediment accumulation rate and charcoal counts
